# Minhas configurações padrões
---

Antes de usar altere o arquivo **configs.sh**

## Como usar:

- Abra o terminal e cole: `curl -L https://github.com/paulopotter/dotfile/archive/master.zip`
- Descompacte o zip
- `cd dotfile`
- Altere o arquivo **configs.sh**
- `./install_osx`
